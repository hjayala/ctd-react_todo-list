## Creating React Project with Template

This repository was creating using the following command originally: 

`npx create-vite@latest --template react .`

Then, dependencies were installed/confirmed using:

```
npm install
```

## Starting the Dev Server

In order to start the development server, the following command is to be used:

```
npm run dev
```

The port (default 5173) can be specified using the following command:
```
npm run dev -- --port 8080
```

Then, the live view is accessible at:

```
http://localhost:5173
```
